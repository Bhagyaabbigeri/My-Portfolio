# 🌌 Bhagyashree Reddy — Cinematic Scrollytelling Portfolio

Welcome to my personal scrollytelling portfolio. This project is a high-performance, cinematic web experience built with **Next.js**, **Framer Motion**, and **Canvas** to showcase my work in Full-Stack Development, AI, and Space Technology.

## 🚀 Experience the Journey
The portfolio features a frame-by-frame scroll animation system that takes the user through a cinematic space-themed narrative.

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Animations:** Framer Motion (Scroll-linked animations)
- **Performance:** Optimized HTML5 Canvas for frame sequence rendering
- **Responsiveness:** Fully mobile-responsive with custom hamburger menu and adaptive layouts

## 🏗️ Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Bhagyaabbigeri/My-Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
